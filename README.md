# ToDo Apps
Create a ToDo Application where we can create, update, read, delete a task (ToDo), with status and deadline.

ToDo Apps must be able to invite one or more person, in spite of the person are already registered or new member.

### Features & Tech
* jQuery
* oauth 
* login with userName or email and password
* CRUD ToDo
* CRUD CollaborationToDo
* Invite a person or more for collaborate
